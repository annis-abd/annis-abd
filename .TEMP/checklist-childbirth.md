
# 🍼 รายการเตรียมของสำหรับลูกแรกเกิด

## 👶 ของใช้สำหรับลูก

### เสื้อผ้า
- [ ] เสื้อแขนสั้น / แขนยาว (ผูกหน้า) 5–7 ตัว  
- [ ] กางเกงขายาว / ขาสั้น 3–5 ตัว  
- [ ] ถุงมือ ถุงเท้า หมวก  
- [ ] ผ้าอ้อม (ผ้าสาลูหรือผ้าสำลี) 10–15 ผืน  
- [ ] ผ้าห่อตัว 3–5 ผืน  

### ของใช้ทำความสะอาด
- [ ] สำลี / คอตตอนบัด (แบบปลอดเชื้อ)  
- [ ] ผ้าเช็ดทำความสะอาด (เปียกสำหรับเด็ก)  
- [ ] อ่างอาบน้ำ  
- [ ] ผ้าขนหนูผืนใหญ่ 2–3 ผืน  
- [ ] ผ้าขนหนูผืนเล็ก 4–6 ผืน  
- [ ] สบู่เหลวหรือแชมพูอ่อนสำหรับเด็ก  
- [ ] เบบี้ออยล์ / โลชั่น  
- [ ] แปรงผมเด็กอ่อน  
- [ ] กรรไกรตัดเล็บเด็ก  
- [ ] ที่ดูดน้ำมูก  
- [ ] เทอร์โมมิเตอร์  

### การให้นม
- [ ] ขวดนม 4–6 ขวด  
- [ ] จุกนมสำรอง  
- [ ] เครื่องนึ่งขวดนม / หม้อต้ม  
- [ ] แปรงล้างขวด  
- [ ] ผ้าเช็ดไหล่  
- [ ] ผ้าคลุมให้นม  
- [ ] เครื่องปั๊มนม  
- [ ] ถุงเก็บน้ำนม
