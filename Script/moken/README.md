# Moken

JSON Web Tokens generator for MO project.


## Setup 

1. Install [Python](https://www.python.org/downloads/)
2. Install PyJWT
   ```bash
   pip install PyJWT
   ```
3. Run **moken.exe**


## Build

1. Install pyinstaller
   ```bash
   pip install pyinstaller
   ```
2. Run Build
   ```bash
   pyinstaller --onefile --name moken --distpath . moken.py
   ```
